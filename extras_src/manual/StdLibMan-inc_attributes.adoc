This file contains all the custom attributes definitions used for substitutions
in the StdLib Manual source files.

Defining links as attributes helps to keep the source docs cleaner to read and
work on. Also, in the future we might want to defined some links conditionally,
depending on the output backend (suppressing links in the PDF, or providing
alternative links on the GH Pages website, etc.).
--------------------------------------------------------------------------------
////
